# skillet

A skillet full of agent skills. Production-tested workflows for auditing, improving, and shipping repos with AI coding agents.

Part of the [escoffier-labs](https://github.com/escoffier-labs) kitchen, alongside [brigade](https://github.com/escoffier-labs/brigade) (agent memory and handoffs) and the [solos-cookbook](https://github.com/escoffier-labs/solos-cookbook) (the patterns these came from).

Full README with install instructions lands with the first release. Scaffold in progress.
